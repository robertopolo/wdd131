Site Plan: Online Store

Site Name

StoreEase

Elegí este nombre porque combina las palabras "store" (tienda) y "ease" (facilidad), transmitiendo la idea de una experiencia de compra simple y conveniente. Es fácil de recordar y tiene un aire profesional y moderno.

Dominio opcional disponible: storeease.shop

Site Purpose

El propósito del sitio web es proporcionar una plataforma digital para una tienda de productos generales, mostrando su catálogo de productos, fotos del local físico, y ofreciendo un formulario de contacto para pedidos o consultas. Este proyecto servirá como modelo para ofrecer servicios web a pequeños negocios locales.

Scenarios

  • ¿Qué productos ofrece la tienda y cuáles son sus precios?
  • ¿Cómo puedo ponerme en contacto con la tienda para hacer un pedido?

Color Schema

  • Primario: #3e7cb1 – usado en encabezados, botones y acentos.
  • Secundario: #f5f5f5 – usado como color de fondo para dar limpieza y claridad.
  • Texto: #333 – color principal del texto para buena legibilidad.

Typography

  • Heading Font: Playfair Display – usado para títulos y encabezados importantes.
  • Body Font: Roboto – usado para el texto general del sitio.
